# Слой общих модулей

Содержит максимально переиспользуемые сегменты.

Не имеет слайсов, а только сегменты:

1. UI – компоненты (UI kit)
2. Model – взаимодействие с state (селекторы, экшены) или базовая типизация
3. Lib – вспомогательные функции (хуки и иные)
4. Config – конфигурации (i-18n)
5. API – настройка работы с сетью (axios-instance, react-query-instance)
6. Consts – константы

Добавление обосновано, если данный сегмент можно перенести в другое приложение без привязки к его логике.

**Какие слои могут использовать:** - только элементы данного уровня

**Примеры:** кнопки, модальные окна, хелперы, не привязанные к бизнес логике.
