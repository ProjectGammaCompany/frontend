# Слой страниц

Содержит слайсы, которые являются страницами.

**Правило:** по возможности должна содержать минимальное количество бизнес-логики, запросов и прочего. Всё описанное должно быть в нижележащих по иерархии слоях. В идеале должна быть перечислением фичей и виджетов, обёрнутым в layout.

**Какие слои могут использовать:** widgets, features, entities, shared.

**Примеры:** страница авторизации.
